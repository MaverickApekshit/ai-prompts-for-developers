# AI Prompts & Workflows for Builders

This repository contains practical AI prompts and workflows
for developers, builders, and solo operators.

The focus is on:
- Debugging
- Code generation
- Refactoring
- Productivity workflows
- Thinking in systems, not hacks

## Example Prompts

### Debugging
Explain this error, identify the root cause, and suggest fixes:

[ERROR_LOG]

### Code Refactoring
Refactor this code for readability and performance:

[CODE]

### System Design
Break this requirement into a clean technical design:

[REQUIREMENT]

### Productivity
Turn this repeated task into a reusable system:

[TASK]

### Decision Making
Help me choose between these options based on constraints:

[OPTIONS + CONSTRAINTS]

## How to use
1. Pick a prompt
2. Paste into your AI tool (ChatGPT, Claude, Gemini, Cursor)
3. Replace placeholders
4. Iterate
