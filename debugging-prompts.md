# Debugging Prompts for Developers

These prompts help analyze errors, identify root causes, and fix bugs faster using AI tools like ChatGPT, Claude, Gemini, or Cursor.

---

## 1. Error Analysis Prompt

Explain this error message and identify the most likely root cause.

Then propose 3 possible fixes and explain the tradeoffs for each.

Error:
[PASTE ERROR MESSAGE HERE]

---

## 2. Bug Reproduction Prompt

Help me reproduce the following bug.

Provide:

• possible causes  
• step-by-step reproduction steps  
• debugging strategy  

Bug Description:
[DESCRIBE BUG HERE]

---

## 3. Debugging Deep Dive

Analyze this issue in detail.

Break it down into:

• potential root causes  
• how to validate each cause  
• recommended fix  
• edge cases to consider  

Context:
[PASTE CODE / ERROR / SYSTEM CONTEXT]
